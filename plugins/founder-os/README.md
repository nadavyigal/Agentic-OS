# Founder OS

Evidence-backed operating workflows for a solo founder managing multiple products.

This private plugin routes to the canonical Agentic OS prompts and files. It does not duplicate product status, credentials, or external app configuration. Connected GitHub, PostHog, Gmail, and Google Calendar plugins remain separate permission boundaries.

## Included skills

- Founder morning: refresh evidence and choose one action.
- Portfolio review: run the CEO or COO workflow at the right altitude.
- Activation read: measure real-user funnels with the exclusion contract.
- Finance review: produce the monthly evidence-backed CFO readout.
- Portfolio Site: refresh sanitized audience contracts and privately publish Portfolio HQ.
