# ERRORS.md Entry Template

Use this format when appending to `tasks/ERRORS.md` or `~/.claude/ERRORS.md`.

```
## YYYY-MM-DD — [Brief title of the failed approach]
Project: [project name, or "global" if cross-project]
Tried: [what was attempted]
Why it failed: [root cause or symptom]
Next time: [what to do instead]
```
