# Clarity Funnel — frozen demo templates

Canned outputs for hackathon demos when live model runs are risky or slow.

| File | Profile | Demo input |
|------|---------|------------|
| `everyday-team-ai-strategy.md` | Everyday | Team AI strategy / tool confusion |
| `builder-team-ai-strategy.md` | Builder | Same input, work-packet emit |

## On-stage use

1. Paste **demo input** from landing page (live or copied).
2. If live skill is slow, open the matching template and scroll through CLARIFY → PROMPT → FINISH.
3. Say: "Same funnel every time; templates show the contract."

## Regenerate

After changing `profiles/*.md`, update these files manually or re-run Clarity Funnel on the demo input and replace contents.
