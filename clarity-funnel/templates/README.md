# Clarity Funnel — frozen demo templates

Canned **session walkthroughs** for hackathon demos when live model runs are risky.

| File | Profile | Shows |
|------|---------|--------|
| `everyday-team-ai-strategy.md` | Everyday | START → CHECK → WRAP (3 acts) |
| `builder-team-ai-strategy.md` | Builder | Same arc + validation / work packet |

## On-stage use

1. Paste **demo input** from landing page.
2. Narrate: "First we lock direction — not a plan dump."
3. Scroll Act 1 → simulate chat → Act 2 CHECK → Act 3 WRAP.
4. Or run live skill and compare to template.

## Regenerate

After changing `profiles/*.md`, re-run Clarity Funnel on the demo input through all three acts and replace file contents.
