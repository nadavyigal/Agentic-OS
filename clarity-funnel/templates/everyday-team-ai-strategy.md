# Frozen demo output — Everyday profile

**Input:**

> Help me create a strategy for my team to start using AI better at work. We tried a few tools but people are confused and I want a clear plan.

---

## CLARIFY

**What you actually want (one sentence):**  
A simple 30-day plan so your team uses AI consistently on real work, without tool chaos or another unused pilot.

**What this is not:**
- Not a company-wide "AI transformation"
- Not a tool comparison essay
- Not training slides nobody uses

**Pick a focus (optional):**
- **A)** One shared workflow everyone learns (e.g. draft → review → send)
- **B)** Fix confusion: which tool for which task
- **C)** Manager rollout: rules, examples, and how to measure adoption

*If you skip this, I'll use **A**.*

---

## PROMPT (copy this)

```text
You are helping a team lead create a practical plan for better AI use at work.

Context:
- Team already tried a few AI tools; people are confused about when to use what.
- Goal: a clear, actionable plan — not a strategy deck.
- Time horizon: 30 days.
- Audience: mixed skill levels; assume no one is an AI expert.

Deliver in this exact structure:

1. ONE-SENTENCE GOAL — what success looks like in 30 days
2. STARTING POINT — 3 likely reasons for confusion (no blame)
3. THE ONE WORKFLOW — pick ONE repeatable workflow the whole team will learn first
   (example pattern: capture context → draft with AI → human review → publish/send)
4. TOOL RULES — simple "use X for Y" table (max 3 tools)
5. WEEK-BY-WEEK PLAN — 4 weeks, max 3 bullets per week
6. TEAM RULES — 5 short rules (privacy, verification, when NOT to use AI)
7. HOW WE'LL KNOW IT'S WORKING — 3 observable signals (not vanity metrics)
8. MANAGER SCRIPT — 5 sentences to say in the next team meeting to launch this

Constraints:
- Plain language, no jargon.
- No vendor pitch.
- No "AI council" or long training program.
- If information is missing, state assumptions explicitly, then continue.

First question before you write the plan: ask me ONLY these two things:
(1) team size and roles (rough), (2) top 2 tasks they do every week that could use AI.
```

*Stay focused on: a 30-day team plan for practical AI use. If the reply adds tools, committees, or "digital transformation," say: "Narrow to the one workflow and 30-day plan we defined."*

---

## FINISH

**At a glance**
- Week 1: Pick one workflow + tool rules
- Week 2: Pilot with volunteers (3 people)
- Week 3: Team-wide + office-hours slot
- Week 4: Retrospective + keep or cut tools
