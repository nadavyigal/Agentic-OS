# Prompt: Update Lessons

```txt
Review whether a new lesson should be saved.

Save it only if it is repeated, costly, easy to forget, or likely to matter again.

If saving:
1. Add it to LESSONS.md.
2. Convert it into a short reusable rule.
3. If project-specific, update that project's local tasks/lessons.md.
4. If workflow-changing, update the relevant workflow or prompt.

Do not save noisy one-off notes.
```
