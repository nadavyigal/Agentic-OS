# Prompt: Create Project Bridge

```txt
Create a global project bridge file for a product.

Do not copy full local docs.

Include:
- Product role in the global portfolio
- What the project is the source of truth for
- How it should connect to other projects
- What it should not inherit from other projects
- Current priorities
- QA/release expectations
- Local files agents should read inside the repo
- Open bridge questions

Keep it short enough to load selectively.
```
