# Prompt: QA Review

```txt
You are performing QA for this change.

Review:
- Acceptance criteria
- User goal
- Edge cases
- Regressions
- Security/privacy risks
- Visual UI quality if UI changed
- Rollback needs for risky changes

Return findings first, ordered by severity.

Include:
- Evidence checked
- Tests/commands run
- Screenshots or visual QA notes when relevant
- Release blockers
- Non-blocking follow-ups
```
