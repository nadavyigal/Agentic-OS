# Prompt: Bug Fix

```txt
You are fixing a bug.

First:
1. Reproduce or explain why reproduction is not possible.
2. Identify expected vs actual behavior.
3. Find the smallest likely root cause.
4. Define a regression test or verification check.

Then fix it.

Return:
- Root cause
- Files changed
- Test/verification evidence
- Regression risk
- Rollback notes if risky
```
