# Prompt: Plan Feature

Use inside a project repo after reading the local OS files.

```txt
You are planning a feature for this repo.

First read:
1. AGENTS.md
2. CODEX.md or CLAUDE.md
3. tasks/lessons.md
4. The relevant local workflow/spec files
5. The global bridge file only if cross-project context is needed

Do not jump into code.

Produce:
- User goal
- Problem statement
- Proposed solution
- Non-goals
- Risks and assumptions
- Small development stories
- Test/QA plan
- Files likely involved
- Rollback notes if risky
```
