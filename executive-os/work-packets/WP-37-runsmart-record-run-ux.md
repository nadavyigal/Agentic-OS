### Explicitly deferred (with reason)

- **Live calories/steps**: benchmark says not table stakes ‚Äî Strava's live run stats are time/distance/pace/splits and Apple's iPhone-only Outdoor Run shows time/distance/pace (calories require Watch/HR hardware). A phone-GPS calorie estimate would be low-trust filler and contradicts the calm-coach taste profile. Revisit only with HealthKit/pedometer integration and a real accuracy story.
- **Live per-km splits** (Strava 2025 parity): real feature work, not polish; depends on S5's real-split computation landing first.
- **Elapsed vs moving time display**: single moving "Time" is acceptable at v1; rename the label to "Moving time" opportunistically inside S3's diff if trivial.
- **Screen-awake toggle, lock-screen/Live Activity controls, haptic pass, Dynamic Type/VoiceOver audit**: stretch items from the audit's optional list; none block trust.
- Delete-dialog Garmin copy + `timeLabel` "90:00" for >1h runs: bundle into whichever story touches the file last, if free.
