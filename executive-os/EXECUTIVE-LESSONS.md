# Executive Lessons

Reusable executive lessons only. These should improve future CEO, CFO, or Analysis OS
behavior. Do not save noisy details, one-off preferences, or anything that belongs in
a local project `tasks/lessons.md`.

## When To Save A Lesson

Save when a lesson is repeated, expensive, easy to forget, relevant across products,
and likely to improve future executive decisions.

## Cross-Layer Rule

If a lesson affects local product execution, also recommend updating that project's
local `tasks/lessons.md`. If it changes a rule or workflow, update the relevant file
in `executive-os/` or the global rule files.

## Lesson Format

- **Context** — when it happened.
- **Issue** — the mistake or expensive surprise.
- **Reusable rule** — the one-line takeaway.
- **Applies to** — which sub-OS / products.
- **Follow-up** — files updated as a result.
